# Testing Web Apps with Cypress

## Set Up

### Install & Run

- npm i cypress --save-dev
- npx cypress open
- OR package.json -> "cypress": "cypress open" -> npm run cypress
- OR npx cypress run (for CLI) -> package.json -> "cy:run": "cypress run" -> npm run cy:run
