## Basic testing process

1. Tab/keyboard navigation
   - interactions
   - focus
   - traps
   - common key commands (arrows, escape, space, enter, etc.)
2. Axe, WAVE, Lighthouse, etc.
3. Zoom (200%)
4. Test othe visual modes
   - light/dark mode
   - high contrast
   - reduced motion
5. Screen readers
6. Check for missing trancripts, captions, etc.
